Chenarul unei matrici. Iterarea în spirală a unei matrici.

Trebuie precizat că iterarea printr-o matrice se poate face în tot felul de moduri, iar unele probleme vor cere o iterare ciudată. Una dintre acestea este iterarea prin chenarul unei matrici. Mai întâi trebuie sa observăm ce înseamnă chenarul unei matrici, ca apoi să putem itera prin ea. Prin iterarea chenarul unei matrici, putem introduce și iterarea în spirală.

Observăm în această matrice faptul că marginea de sus se află pe linia 0, marginea din dreapta pe coloana 6, marginea de jos pe linia 3, iar marginea din stânga pe coloana 0.

Astfel, putem generaliza ca marginea să fie reuniunea tuturor elementelor de pe prima și ultima linie, precum și prima și ultima coloană.

Codul următor va realiza suma tuturor numerelor de pe marginile matricei.

Cod exemplu:

int v[1010][1010], n, m, i, j, k;
(citire)
///Indexam de la 1
i = 1;
for (j = 1; j < m; j++) ///mergem până la ultimul element, fără a-l aborda, deoarece îl vom aborda în următorul for.
    k = k + v[i][j];
j = m;
for (i = 1; i < n; i++)
    k = k + v[i][j];
i = n;
for (j = m; j > 1; j--)
    k = k + v[i][j]
j = 1;
for (i = n; i > 1; i--)
    k = k + v[i][j];
cout << k;

Pentru a merge în spirală, putem adăuga un alt for, cu indicele k, care pleacă de la 0 și va fi mai mic decât min(n,m)/2. De fiecare dată când terminăm o iterare pe o margine, k crește, astfel scăzând mărimea matricei, permițându-ne să continuăm iterare pe submatricea nouă, creând spirala.

Cod exemplu:

int v[1010][1010], fin[1010*1010], lg, n, m, i, j, k;
(citire)
for (k = 0; k < min(n,m)/2; k++)
{
    ///Indexam de la 1
    i = k+1;
    for (j = k+1; j < m-k; j++) ///mergem până la ultimul element, fără a-l aborda, deoarece îl vom aborda în următorul for.
    fin[++lg] = v[i][j];
    j = m-k;
    for (i = k+1; i < n-k; i++)
        fin[++lg] = v[i][j];
    i = n-k;
    for (j = m-k; j > k+1; j--)
        fin[++lg] = v[i][j];
    j = k+1;
    for (i = n-k; i > k+1; i--)
        fin[++lg] = v[i][j]; 
}
if (min(n, m)%2 == 1) ///caz special
    for (i = k+1; i <= n-k; i++)
        fin[++lg] = v[i][j];
for (i = 1; i <= lg; i++)
    out << fin[i] << ‘ ‘; ///afișăm matricea în spirală.
