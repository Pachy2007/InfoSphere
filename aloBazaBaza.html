Trecerea unui număr dintr-o bază în alta

Considerăm o bază ca fiind de ordinul x daca are exact x cifre. Astfel, baza 2 are doar cifrele (0, 1), baza 8 are (0, 1, 2, 3, 4, 5, 6, 7), baza 10 are (0, 1, ..., 8, 9, 10), iar baza 16 are (0, 1, ..., 8, 9, 10, A, B, C, D, E, F). Oamenii sunt obișnuiți cu baza 10, deoarece aceasta este uzual vieții de zi cu zi. În schimb, calculatoarele se utilizează principal de baza 2, astfel fiind uzual pentru ele această bază.

Pentru a trece un număr din baza 10 în baza y, se efectueaza următorii pași:

pow10 = 1;
            while (n != 0)
            {
    new_n = new_n + n % y * pow10;
    pow10 = pow10*10;
    n = n / y;
}

Pentru a trece un număr din baza y în baza 10, se efectuează următorii pași:

cin >> b >> n;
while(n)
{
    n--;
    int x; cin >> x;
    new_n = new_n * b + x;
}

Pentru a trece un număr din baza x în baza y, se trece numărul din baza x în baza 10, apoi din baza 10 în baza y.
