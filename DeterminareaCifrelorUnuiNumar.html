Prelucrarea cifrelor unui număr

Cifrele unui număr pot fi manipulate prin utilizarea restului și a câtului la împărțirea la o putere de 10. Datorită acestui fapt, noi putem aplica diferite operații pe numere:

Determinarea cifrelor unui număr

Să ne gândim la un număr (natural), n = 274. Prin restul împărțirii la 10, obținem cifra 4, care este ultima cifră al lui n. Mai observăm faptul că, dacă împărțim la 10 numărul, putem elimina ultima cifră.

Dezvoltând ideea anterioară, putem obține orice cifră a oricărui număr relativ ușor. Pentru exemplul de mai sus, restul împărțirii la 100 este 74. Pentru a obține a doua cifră de la stânga la dreapta, putem aplica o împărțire la 10 din 74, astfel obținând 7.

O altă metodă de a determina a doua cifră este prin a împărți la 10, apoi prin a obține restul împărțirii la 10. Împărțind 274 la 10 obținem 27, iar restul la împărțirea cu 10 este 7.

Generalizând, putem obține orice cifră a oricărui număr prin următoarea instrucțiune:

cif = (n / pow(10, c-1)) % 10;

unde cif este variabila dorită, n este numărul nostru, c este cifra de la dreapta la stânga pe care o dorim, iar pow() este o funcție ce returnează puterea unui număr la oricare exponent.

Observând această instrucțiune, noi putem elimina sau adăuga orice cifră dintr-un număr, prin a ține minte restul unui număr și printr-o împărțire.

Pentru adăugare, dacă dorim să adăugăm cifra 3 la prima poziție de la stânga la dreapta a numărului 274, am reține restul la împărțirea la 100, care este 4. După împărțim cu 100, obținând n = 2. Înmulțim cu 10 și adăugăm cifra pe care dorim să o adăugăm, obținând 23. Înmulțim n cu 100 și adăugăm restul pe care l-am reținut, obținând 2374. Un cod ar fi: 


aux = n % pow(10, c);
n = n / pow(10, c);
n = n * 10;
n = n + x;
n = n * pow(10, c);
n = n + aux;

în care n este numărul, x este cifra pe care o adăugăm, c este poziția pe care inserăm numărul de la dreapta la stânga, aux este variabila unde ținem minte restul, iar pow() este o funcție ce returnează puterea unui număr la oricare exponent.

Pentru scădere, am efectua aproape aceleași operații. Pentru a elimina a doua cifră de la stânga la dreapta a numărului 2374, am reține restul la împărțirea cu 100, am împărți numărul cu 1000, după am înmulți cu 100 și am adăuga restul. Un cod ar arăta astfel:

aux = n % pow(10, c-1);
n = n / pow(10, c);
n = n * pow(10, c-1);
n = n + aux;

în care n este numărul, c este poziția pe care inserăm numărul de la dreapta la stânga, aux este variabila unde ținem minte restul, iar pow() este o funcție ce returnează puterea unui număr la oricare exponent.

Știind acum cum putem să determinăm oricare cifră a unui număr, observăm faptul că noi putem itera asupra fiecărei cifre dintr-un număr. Acest lucru ne permit să aplicăm diferite operații pe cifrele numerelor. Un exemplu de cod care iterează asupra cifrelor este:

int n, cif; 
cin >> n;
do              
{               
    cif = n % 10; ///obținem cifra.
    n = n / 10; ///eliminăm ultima cifră.
}while(n != 0);

în care n este numărul iar cif este cifra dorită.

Observați utilizarea funcției do…while() în locul funcției while(). O eroare făcută foarte des în rândul elevilor este neabordarea situației n = 0, caz în care nu se execută nimic, rezultând în puncte pierdute . Do…while() evită această situație.

Niște prelucrări pe care le putem aplica asupra lui n sunt: oglindirea, suma cifrelor, eliminarea cifrelor impare și dublarea cifrelor pare.

Vom prezenta doar oglindirea și eliminarea cifrelor impare, restul prelucrărilor fiind lăsate pentru ca cititorul să le rezolve.

int n, oglx = 0;
cin >> n;
do
{
oglx = oglx * 10 + n%10;
n = n / 10;
} ///oglinditul unui nr.
            
            
int n, put10 = 1, x = 0;
cin >> n;
do
{
if (x % 2 == 0) //este restul împărțirii la 2 = 0? Dacă da, este par.
x = x * put10 + n % 10;
x = x / 10;
put10 = put10 * 10;
}
n = x; ///eliminarea cifrelor impare ale unui nr.
