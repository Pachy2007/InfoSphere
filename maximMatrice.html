Maximul/minimul dintr-o matrice

Maximul și minimul dintr-o matrice se poate determina super ușor, prin iterarea acesteia și reținerea a două valori, maxi și mini, care vor fi maximul și minimul. În cazul în care elementele din matrice pot fi negative, atât maximul, cât și minimul vor trebui să fie definite. Acestea pot fi definite fie ca o valoare extrem de mică, sau pot fi definite ca fiind prima valoare din matrice: m[1][1] (sau m[0][0] dacă matricea este indexată de la 0)

Cod exemplu:

int mini, maxi, v[1010][1010], n, i, j, m;
//citire
mini = maxi = v[1][1];
for (i = 1; i <= n; i++)
    for (j = 1; j <= m; j++)
    {
        mini = min(mini, v[i][j]);
        maxi = max(maxi, v[i][j]);
    }
cout << mini << ’ ’ << maxi;
