Declararea. Citirea. Afișarea.

Definim Vectorii: Vectorii sunt tablouri de elemente omogene, care pot sa aibă o dimensiune fixă sau dinamică. Vectorii despre care vom discuta au o dimensiune fixă. Vectorii sunt utilizați pentru a reține un tablou de elemente pentru prelucrare după citire. Dacă cumva nu este necesară o astfel de prelucrarea, vectorii nu sunt necesari!

Declararea unui vector se face astfel: 

tip_date nume_vector [dimensiune];
Exemplu: int v[100];

Citirea unui vector se face prin a citi un număr n, care este dimensiunea vectorului. Cod Exemplu: 

cin >> n;
for (i = 1; i <= n; i++) ///vectorii pot fi indexati de la oricare i. Personal recomand indexarea de la 1, deoarece este mai ușoară de înteles și evită anumite cazuri speciale la mulți algoritmi.
{
    cin >> v[i];
}

Afișarea unui vector se face exact la fel ca citirea vectorului. Cod Exemplu: 

for (i = 1; i <= n; i++)
cout << v[i];
