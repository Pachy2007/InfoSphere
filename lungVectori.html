Cea mai lungă secvență crescatoare

Pentru a determina cea mai lunga secvență crescătoare, vom ține minte lungimea curentă și vom itera prin vectorul v, uitându-ne constant în spate să vedem dacă secvența pe care o continuăm este încă crescătoare. Dacă nu, verificăm dacă lungimea curentă este lungimea maximă, și resetăm lungimea curenta la a fi 1.

Cod exemplu: 

int n, v[1010], lg = 1, best = 0;
///citire
for (int i = 2; i <= n; i++)
{
    if (v[i] == v[i-1])
    { lg++; continue; }
    best = max(best, lg);
    lg = 1;
}
