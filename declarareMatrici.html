Declararea. Citirea. Afișarea.

Definim matricea ca fiind un tablou bidimensional de valori omogene. Matricea poate fi imaginată ca fiind un plan 2D de lungime n și lățime m, precum o hartă care este împărțită în celule.

Declararea unei matrici se face prin a preciza tipul datelor, numele și dimensiunea acesteia, astfel:

tip_date nume_matrice [dimensiune_1][dimensiune_2];

Exemplu:

int m[1010][1010];

Observați că mărimea pe care o ocupă o matrice este de dimensiune_1 * dimensiune_2. Aveți grijă la acest fapt, deoarece dimensiunea este o cauză problematică a matricelor!

Citirea și afișarea unei matrici se face similar ca la vectori, doar că avem un pas în plus. Ca să putem citi sau să afișăm dintr-o matrice, trebuie mai întâi să ne setăm o linie pe care vom face operații. Astfel, observăm că citirea și afișarea este identică ca la un vector.

Cod Exemplu:

int m[1010][1010], i, n, m, j;
for (i = 1; i <= n; i++)
    for (j = 1; j <= m; j++)
        cin >> m[i][j]; 
for (i = 1; i <= n; i++)
    for (j = 1; j <= m; j++)
        cout << m[i][j];
