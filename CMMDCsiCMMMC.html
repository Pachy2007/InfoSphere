CMMDC și CMMMC

Definim CMMDC (Cel mai mare divizor comun == Greatest Common Divisor, abreviat „gcd”) ca fiind cel mai mare divizor comun a două numere. Gcd are următoarele proprietăți:

  • a|c și b|c, unde c = gcd(a, b)
  • fie x un număr. Gcd(a, Gcd(b, c)) = Gcd(Gcd(a, b), c)
  • a|bc si Gcd(a, b) = 1, deci a|c
  • a|c, b|c și Gcd(a, b) = 1, deci ab|c
  • Gcd(a, b) = 1, Gcd(a, c) = 1, deci Gcd(a, bc) = 1;
  • Gcd(a, 0) nu se definește, deoarece 0 nu are divizori.

Există 2 metode prin care se poate determina Gcd: Algoritmul lui Euclid prin scăderi: Algoritmul lui Euclid cu scăderi se bazează pe ideea că cele mai mare divizor a două numere divide și diferența acestora. Algoritmul va scădea numărul mic din cel mare până când cele două numere devin egale. Valoarea acestora este Gcd(a, b). Exemplu Gcd(25, 15): 

25, 15 – scădem 25-15 = 10
10, 15 – scădem 15 – 10 = 5
10, 5 – scădem 10 – 5 = 5
5, 5 – stop.

Un cod exemplu ar fi:

int Gcd(int a, int b)
{
    if (a == 0 || b == 0) return -1; ///nu se defineste Gcd(a, 0);
    while (a != b) {
    if (a > b) swap(a, b);
    a -= b;
    }
    return a;
}

Algoritmul lui Euclid prin împărțiri: Algoritmul lui Euclid cu împărțiri se bazează pe ideea că cel mai mare divizor a două numere divide și restul împărțirii acestora, conform teoremei împărțirii cu rest. Algoritmul va determina restul împărțirii lui a la b (pentru care b < a). Apoi, a devine b, iar b devine restul împărțirii. Algoritmul se va executa până când m devine 0, iar rezultatul este a. Exemplu Gcd(25, 15):

25, 15 – restul = 10, a = 15, b = 10.
15, 10 – restul = 5, a = 10, b = 5.
10, 5 – restul = 0, a = 5, b = 0.
5, 0 – stop.

Un cod exemplu ar fi:

int Gcd(int a, int b)
{
if (a == 0 || b == 0) return -1; ///nu se defineste Gcd(a, 0)
if (a < b) swap(a, b);
    while(b != 0)
    {
        int r = a % b.
        a = b; 
        b = r;
    }
    return a;
}
