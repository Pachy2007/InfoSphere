Șirul lui Fibonnaci

Șirul lui Fibonnaci este definit astfel: un șir ai cărui primele 2 elemente sunt 1. Fiecare din următoarele elemente sunt suma elementelor anterioare. Elementele din șirul lui Fibonacci se determină pe care brută. Un cod ar arăta astfel:

int a = 1, b = 1, c, n;
cin >> n;
while(n)
{
    --n;
    c = (a + b);
    cout << c << ‘\n‘;
    a = b; b = c; ///numerele cresc aproximativ exponential! Astfel, dupa n ~= 30, numerele pot iesi din tipul int!
}
