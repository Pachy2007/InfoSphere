Interclasarea unui vector

Interclasarea este un algoritm ce combină 2 vectori sortați într-unul singur, de complexitate n + m, unde n și m sunt dimensiunile celor doi vectori. Acesta se realizează prin rularea în paralel prin cei doi vectori utilizând 2 indici diferiți și inserarea elementelor într-un vector final.

Cod exemplu:

int n,a[100000], m , b[100000], p, c[200000];
int i = 1 , j = 1;
p = 1;
while(i <= n && j <= m)
if(a[i] < b[j])
    c[p ++] = a[i ++];
else
    c[p ++] = b[j ++];
while(i <= n)
    c[p ++] = a[i ++];
while(j <= m)
    c[p ++] = b[j ++];
