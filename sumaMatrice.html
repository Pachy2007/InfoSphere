Suma /produsul/minimul/câte elemente impare pe linia și coloana K

Pentru a afla suma, produsul, minimul, și câte elemente impare pe linia k, este suficient cât să setăm indicele i ca fiind k.

Pentru a afla aceleași lucruri pe coloana k, trebuie să setăm indicele j ca fiind k, și să iterăm prin i-uri.

Cod exemplu:

int v[1010][1010], m,n,i,k,j;
//citire
cin >> k;
i = k;
for (j = 1; j <= m; j++)
    ///operații.
j = k;
for (i = 1; i <= n; i++)
    ///operații.
