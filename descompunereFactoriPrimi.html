Descompunerea în factori primi

Descompunerea în factori primi se realizeaza într-o manieră similară ca la determinarea divizorilor unui număr, doar că cu anumite modificări. Observăm faptul că primul divizor este întotdeauna prim, iar prin a elimina apariția acestuia din număr prin împărțiri, multiplii acestuia nu vor mai fi luați niciodată în considerare, asigurându-ne că următorul divizor este prim. O optimizare pe care o mai putem face este observația că nu vom considera niciodata multiplii lui 2 ca divizori. Din acest motiv, putem să înjumătățim timpul de executare a programului printr-o optimizare. Astfel, avem acest algoritm:

int n, q = 2, x = 0;
in >> n;
while (n % q == 0) ///optimizarea, scoatem q == 2 ca fiind caz special.
{
    x++;
    n = n / q;
}
if (x != 0)
    cout << x << ‘ ‘ << q << ‘\n‘;
for (q = 3; q * q <= n; q=q+2)
    {
    x = 0;
    while (n % q == 0)
    {
        x++;
        n = n / q;
    }
    if (x != 0)
        out << x << ‘ ‘ << q << ‘\n‘;
}
if (n != 1) //daca n nu este 1, atunci x-ul rămas este prim.
    cout << 1 << ‘ ‘ << q << ‘\n‘;

O observație este faptul că, prin descompunerea în factori primi, putem să ne dăm seama dacă un număr este prim sau nu. Dacă nu s-a efectuat nicio operație și n nu este 1, atunci n este prim.
