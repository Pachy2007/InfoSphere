Divizorii unui număr

Divizorii se pot afla prin 2 metode. Vom explora fiecare metodă, precum și verificarea daca un nr este prim, afișarea divizorilor și numărarea acestora.

Prima metodă este cea brută, aceea fiind de a lua fiecare număr până la n și a verifica dacă se divid cu acesta. Acest cod va executa n operații. Pentru a fi mai eficient, putem face observația că n nu are divizori mai mari decât n/2. Astfel, codul următor execută n/2 operații:

int n, i;
for (i = 2; i <= n / 2; i = i + 1)
{
if (n % i == 0) 
///i divide n
}

A doua metodă este mai optima. Un divizor al lui n se poate înmulți cu un alt divizor pentru a obține n. Astfel, observăm faptul că cel mai mic divizor dintre acești doi nu este mai mare decât radical din n. Astfel, noul cod va executa doar sqrt(n) operații, unde sqrt(n) (Square Root == rădăcina pătratului) este radical din n. Este important de menționat ca nu vom itera decât numai prin prima jumătate din divizori, astfel cititorul va trebui să aibă grijă la implementări să considere acest fapt. Un cod ar arăta astfel: 

int n, q;
for (q = 2; q * q <= n; q = q + 1)
{
if (n % q == 0)
///q divide n și (n/q) divide n, IMPORTANT, daca q * q == n, atunci q si (n/q) sunt egali, iar n este pătrat perfect.
}
