Condiții pe diagonale in matricile pătratice si condiții pe zone(nord, sud, este,vest)

O matrice pătratică este o matrice care are dimensiunile coloanelor și a liniilor egale. Într-o matrice pătratică, există două diagonale: diagonala principală, și diagonala secundară.

Diagonala principală începe de la v[1][1] și se încheie la v[n][n], și diagonala secundară, care începe de la v[1][n] și se încheie la v[n][1]. Într-o matrice pătratică, exista mai multe diagonale care sunt paralele cu acestea. În general, un cod care creează suma a diagonalei unui pătrat ar arăta astfel:

Cod exemplu:

int v[1010][1010], i, j, t, n, q, a, b, s, tip; //tipul unu este diagonala principală, tipul doi este cea secundară.
//citire
for (t = 1; t <= q; t++)
{
    s = 0;
    cin >> tip >> a >> b;
    if (tip == 1) 
        while ( a <= n && b <= n)
        {
            s = s + v[a][b]; a++; b++;
        }
    else
        while (a <= n && b >= 1)
        {
            s =s + v[a][b]; a++; b--;
        }
    cout << s << ‘\n‘;
}
