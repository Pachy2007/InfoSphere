Construirea unui vector din alt vector. Inversarea unui vector in alt vector

Un exemplu de construire unui vector din alt vector este interclasarea, unde vectorul final este construit din 2 vectori. Vom arăta inversarea unui vector în alt vector, după care vom propune probleme pentru a aplica pe acest subiect.

Pentru inversarea unui vector, vom itera prin vectorul inițial v și vom atrbibui elementul v[i] valorii fin[n-i+1], pentru v indexat de la 1, sau fin[n-i], pentru v indexat de la 0, unde fin este vectorul final.

Cod exemplu:

int n, v[1010], fin[1010];
///citire
for (int i = 1; i <= n; i++)
    fin[n-i+1] = v[i];
